Table cells score model in each product capability from 0 to 100. Higher is better.

  • docs - working with large documents and knowledge bases
  • crm - capabilities for working with product catalogues and marketplaces
  • integrate - ability to interact with external APIs, services and plugins
  • marketing - assistance in marketing activities: brainstorming, idea refinement, text generation
  • reason - ability to reason within the provided context
  • code - code generation and reasoning
  • final - final model score across all categories
  • Cost is estimated cost of running workfload. For local models we take into account GPU hardware and model speed. Not used in scoring.
  • Speed is estimated speed of of the model, in requests per second. Higher is better. Not used in scoring.

Colors are adjusted, based on the best performing score in each category.
